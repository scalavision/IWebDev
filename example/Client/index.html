WebDevExample

To make it work:

  • In jsHelper folder, install dependencies with `yarn` or `npm`, start the node js application
  • Start the IWebDev server through SBT
  • Open this html file in a browser, you should see it connect to the WebDev Servers websocket
  • Start continous compilation of this project with ~pushToClient

Whenever files in this project has changed, the newly compiled / fastOptJS'ed javascript is pushed to this client, and the javascript dom node is replaced with the new javascript

To test the CSS stylesheet update, use the ammonite script located in the CSS folder.
